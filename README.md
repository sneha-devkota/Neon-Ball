# Neon Ball Game

## Overview
This project is a browser-based arcade game where players control a glowing neon ball and navigate through moving obstacles. The goal is to survive as long as possible by passing through pipe gaps while gravity continuously pulls the ball downward. The game tracks the player's score and stores the best score using localStorage.

## Features
- Smooth physics-based jumping mechanics
- Dynamic pipe generation with random gaps
- Increasing game speed as the score increases
- Real-time score tracking
- Persistent high score using localStorage
- Neon glow visual effects and gradient backgrounds

## Technologies Used
- HTML for structuring the game interface
- CSS for styling and UI layout
- JavaScript for game logic and physics
- HTML5 Canvas for rendering the game graphics

## Usage
- Press the **Spacebar** or **click on the canvas** to make the ball jump.
- Avoid hitting the pipes or the top and bottom boundaries.
- Pass through pipe gaps to increase your score.
- The game ends when the ball collides with an obstacle.

## Controls
| Input | Action |
|------|-------|
| Spacebar | Jump |
| Mouse Click | Jump |
